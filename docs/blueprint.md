# **App Name**: TempoWise

## Core Features:

- Time Tracking: Start, pause, and stop timers for different activities, assigning them to categories and tags.
- Automated Suggestions for Labels: Suggest labels and categories automatically based on user input, using AI to predict relevant tags.
- Statistics Generation: Automatically generate graphs and charts for visualizing time spent across days, weeks and months.
- Weekly goal definition: Allow users to define weekly goals for each category, track progress, and send notifications.
- Pomodoro Timer: Set up configurable Pomodoro sessions (e.g., 25/5 minutes) and automatically log time in the selected category.
- Energy Level Logging: Daily tracking of user's energy and motivation levels to correlate with activity patterns. Insights are automatically produced by an LLM tool.
- Google Calendar Integration: Synchronize events from Google Calendar to track time spent on meetings and appointments.

## Style Guidelines:

- Primary color: Light grey-blue (#D1DCE5) for a calm, efficient feel.
- Background color: Off-white (#F8F9FA) to enhance readability and reduce eye strain.
- Accent color: Soft sky blue (#96B4CE) for interactive elements and highlights.
- Body and headline font: 'Inter' for a modern, neutral, and highly readable experience.
- Use clean, line-based icons to represent categories and actions.
- Prioritize a clean and efficient layout with clear hierarchy.
- Subtle animations for user interactions, like starting/stopping timers or transitions between sections.
